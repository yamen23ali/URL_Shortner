*- About : 
This is url shortner application by Yamen Ali based on Yii php framework and MySQL database server .

============================

*- How to use :
In order to use this code , do the follwoing :
1- Create the following table in your DB :
"CREATE TABLE URL_MAPPING ( URL_KEY VARCHAR(50) , URL VARCHAR(1000) );"
2- Copy the files from the repository to the corresponding paths of your Yii application .
3- Have fun :)) .
